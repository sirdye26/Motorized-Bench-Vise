The concept for the automated, motorized bench vise seeks to incorporate a mechanical means for action on the movable jaw. The idea is centered on modification of an existing bench vise, engineering a design to incorporate the use of a motor to actuate the handle that provides forward and reverse action of the screw mechanism. The motor will exist parallel to the bench vise, connected via a jointed linkage. This motor will be controlled using a foot-pedal device, attached to the motor, which offers a forward and reverse directional signal to the actuator. The motor will connect to a common AC 120V outlet, standard in most environments.
